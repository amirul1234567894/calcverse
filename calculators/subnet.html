Subnet Information

Understanding Subnets and CIDR Notation

CIDR (Classless Inter-Domain Routing) notation represents an IP address with its network prefix length. For example, 192.168.1.0/24 means a 24-bit network prefix, giving 254 usable host addresses.

Common CIDR References

/8 = 16,777,214 hosts | /16 = 65,534 hosts | /24 = 254 hosts | /28 = 14 hosts | /30 = 2 hosts | /32 = single host
